## 생성 모델이 소유하면 안 되는 것

자료구조를 배우는 게임에서 모델이 문제와 정답 기준까지 즉석 생성하면 그럴듯하지만 틀린 내용을 가르칠 수 있습니다. 이 프로젝트는 문제와 정답 anchor를 사람이 작성하고, AI는 자유서술 답안을 제한된 등급으로 판정합니다. 실제 provider는 dialogue 설정에 따라 anchor 밖 새 지식을 금지한 한 줄 반응과 표정을 선택적으로 생성하므로 AI가 “채점만 한다”고 표현하지 않습니다.

호감도, 문제 순서, 재설명, 엔딩 같은 상태 전이는 결정론 엔진이 소유합니다. 모델 응답이 늦거나 형식이 틀려도 게임 규칙 자체가 바뀌지 않게 하기 위해서입니다.

## 외부 의존성이 없어도 완주 가능한 구조

정적 환경이나 API 키가 없는 상태에서는 mock grader가 같은 계약을 구현합니다. 실제 모델을 사용하는 서버 프록시는 선택 사항이며 브라우저에 키를 노출하지 않습니다.

검증은 개별 함수 테스트만으로 끝내지 않고 다음 경로를 함께 확인했습니다.

- 25개 엔진 테스트
- 8개 시나리오 beat의 순서
- 오답 후 재설명과 재도전
- mock grader만 사용한 브라우저 완주

## 아직 주장하지 않는 것

이 기록은 학습 효과가 개선됐다는 실험 결과가 아닙니다. 공개 사용자 데이터도 없습니다. 현재 근거로 확인되는 개인 역할은 UI·대화창과 아트 방향이며, 스토리·정답 anchor·엔진·서버의 구현 귀속은 추가 확인 전까지 개인 성과로 표현하지 않습니다.

> 발행 전 체크: 개인 기여 범위를 팀 기록과 다시 대조하고 UI 캡처의 공개 가능 여부를 확인한다.
